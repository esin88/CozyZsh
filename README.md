# CozyZsh
Cozy zsh setup with minimum set of features for comfortable work.

### Features

1. `git` completion
1. case insensitive completion
1. completion based on command history
1. syntax highlight
1. git branch info displayed in prompt
1. `UP` and `DOWN` arrows for command hitory search
1. [Optional] `kubectl` shortcut and completion

### Installation

```
git clone git@github.com:esin88/CozyZsh.git
cd CozyZsh
chmod +x install.sh
./install.sh
```
Don't forget to restart your terminal
